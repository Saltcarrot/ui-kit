# Цвета

#### Основной
* **--buttonPrimaryBGColor** - цвет;
* **--buttonPrimaryHoverBGColor** - цвет при наведении;
* **--buttonPrimaryActiveBGColor** - цвет в состоянии фокуса;
* **--buttonPrimaryTextColor** - цвет текста;
* **--buttonPrimaryDisabledBGColor** - цвет заблокированной кнопки;
* **--buttonPrimaryTransparentHoverBGColor** - цвет при наведении на прозрачную кнопку;
* **--buttonPrimaryTransparentActiveBGColor** - цвет в состоянии фокуса прозрачной кнопки.

* **--buttonSecondaryHoverBGColor** - цвет при наведении;
