# Переменные

#### Основные
* **--buttonBorderWidth** - ширина рамки;
* **--buttonBorderStyle** - стиль рамки;
* **--buttonBorderColor** - цвет рамки.

#### Малая кнопка
* **--buttonSmallGap** - отступ между текстом и иконкой;
* **--buttonSmallPaddingX** - внутренний отступ по горизонтали;
* **--buttonSmallPaddingY** - внутренний отступ по вертикали;
* **--buttonSmallOnlyIconPadding** - внутренний отступ кнопки с иконкой;
* **--buttonSmallFontSize** - размер шрифта текста;
* **--buttonSmallLineHeight** - высота строки текста;
* **--buttonSmallFontWeight** - толщина текста;
* **--buttonSmallBorderRadius** - радиус скругления углов.
